# Properly Nested and Balanced Parentheses

Determine whether a string has properly balanced and properly nested parentheses.

## Example 1:
Input: `"(())"`

Output: `true`

## Example 2:
Input: `"(()(words words words))"`

Ouput: `true`

## Example 3:
Input: `")("`

Ouput: `false`

## Example 4:
Input: `"())"`

Output: `false`
